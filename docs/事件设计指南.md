## 一、核心设计原则

### 1.1 没有正确答案原则

**每个选项都必须有代价，没有完美的选择。**

```
❌ 错误示例：
选项A：清廉正直 → 所有属性+10（太完美）
选项B：贪污受贿 → 所有属性-10（太糟糕）

✅ 正确示例：
选项A：清廉正直 → 清议+15，财帛-20，可能被孤立
选项B：贪污受贿 → 财帛+30，清议-20，可能被弹劾
选项C：明哲保身 → 无变化，但错失机会
```

### 1.2 蝴蝶效应原则

**小改变可能引发大波澜，当前选择影响未来事件。**

```
选择A → 设置flag_A → 影响事件B的触发条件
选择B → 设置flag_B → 影响事件C的选项显示

示例：
1629年选择"结交阉党" → 设置flag:阉党关系
→ 1630年"诛杀魏忠贤"事件中，可能被牵连
→ 1631年"温体仁清算"事件中，可能被清洗
```

### 1.3 历史惯性原则

**改变历史很难，大势难违。**

```
改变历史需要：
1. 高属性（武略>80，理政>70等）
2. 正确时机（特定年份、月份）
3. 前置条件（已触发特定事件）
4. 运气（随机概率）

难度分级：
★ 个人命运：60-80%成功率
★★ 地方事件：40-60%成功率
★★★ 朝中决策：30-50%成功率
★★★★ 历史人物命运：15-30%成功率
★★★★★ 历史大势：5-15%成功率
```

### 1.4 多重后果原则

**一个选择影响多个属性，体现复杂性。**

```
每个选项至少影响2-3个属性：

示例：开仓放粮赈灾
→ 财帛-20（消耗财富）
→ 民望+25（百姓感激）
→ 清议+10（清流赞赏）
→ 道德值+10（良心满足）
→ 设置flag:擅自开仓（可能被弹劾）
```

## 二、属性系统说明

### 2.1 个人能力（玩家可控）

| 属性 | 范围 | 说明 | 提升方式 |
|------|------|------|----------|
| **财帛** | 0-100 | 财富资源 | 俸禄、赏赐、田产、灰色收入 |
| **文韬** | 0-100 | 文学才华、谋略 | 读书、游学、文会、撰写文章 |
| **理政** | 0-100 | 政务处理能力 | 处理政务、完成任务、交际 |
| **武略** | 0-100 | 军事指挥能力 | 边疆任职、军事行动、研读兵书 |
| **体质** | 0-100 | 身体健康 | 休息、养病、药物、补品 |

### 2.2 五方态度（受行为影响）

| 态度 | 范围 | 说明 | 四档评价 |
|------|------|------|----------|
| **圣眷** | 0-100 | 皇帝态度 | >75圣宠 >50受宠 >25平常 ≤25失宠 |
| **中官** | 0-100 | 宦官态度 | >75心腹 >50亲近 >25中立 ≤25敌视 |
| **清议** | 0-100 | 清流评价 | >75东林 >50清流 >25中立 ≤25权臣 |
| **士绅** | 0-100 | 士绅态度 | >75拥戴 >50支持 >25中立 ≤25反对 |
| **民望** | 0-100 | 百姓评价 | >75爱戴 >50称颂 >25平淡 ≤25怨声 |

### 2.3 隐藏属性（决定选择倾向）

| 属性 | 范围 | 说明 | 四档评价 |
|------|------|------|----------|
| **道德值** | 0-100 | 品德操守 | >75君子 >50正直 >25常人 ≤25小人 |
| **欲望值** | 0-100 | 欲望程度 | >75强烈 >50一般 >25淡泊 ≤25清心 |
| **野心值** | 0-100 | 权力野心 | >75勃勃 >50有 >25微 ≤25无 |

### 2.4 五方态度联动规则

```
【联动关系】

圣眷(皇帝) ───┬── 清议(清流) [冲突：皇帝可能不喜欢清流]
              ├── 中官(宦官) [联动：皇帝信任宦官]
              └── 士绅(士绅) [中立：皇帝对士绅态度复杂]

清议(清流) ───┬── 中官(宦官) [冲突：清流反对宦官]
              ├── 士绅(士绅) [联动：清流多出身士绅]
              └── 民望(百姓) [联动：清流关注民生]

中官(宦官) ───┬── 清议(清流) [冲突：宦官被清流反对]
              └── 圣眷(皇帝) [联动：宦官接近皇帝]

士绅(士绅) ───┬── 清议(清流) [联动：士绅支持清流]
              └── 民望(百姓) [冲突：士绅可能剥削百姓]

民望(百姓) ───┬── 清议(清流) [联动：清流关注民生]
              └── 士绅(士绅) [冲突：士绅剥削百姓]
```

**设计时注意：**
- 提升清议时，中官可能下降
- 提升中官时，清议可能下降
- 提升民望时，士绅可能下降（如果涉及减税、赈灾）
- 提升圣眷时，清议可能下降（如果涉及讨好皇帝）

## 三、事件设计流程

### 3.1 确定事件类型

```
1. 历史大事件：20个关键历史节点
2. 地方治理事件：刑名、钱粮、教化、军事
3. 朝堂斗争事件：党争、人事、决策
4. 个人事件：家族、情感、灰色选择
5. 随机事件：根据属性和标记触发
```

### 3.2 确定触发条件

```typescript
conditions: {
  // 时间条件
  year: { min: 1628, max: 1644 },
  month: { min: 1, max: 12 },
  
  // 身份条件
  rank: ["知县", "知府"],
  origin: ["寒门", "诗文清望"],
  
  // 属性条件
  attributes: {
    文韬: { min: 60 },
    武略: { min: 50 }
  },
  
  // 五方态度条件
  gameState: {
    圣眷: { min: 40 },
    清议: { min: 30 }
  },
  
  // 隐藏属性条件
  hidden: {
    道德值: { min: 30 }
  },
  
  // 标记条件
  flags: {
    has: ["前置事件flag"],
    notHas: ["互斥事件flag"]
  },
  
  // 人物存活条件
  charactersAlive: ["袁崇焕"],
  charactersDead: ["魏忠贤"]
}
```

### 3.3 设计选项

**每个事件至少3个选项，最多5个选项。**

```
选项类型：
1. 高风险高回报选项（需要高属性）
2. 低风险低回报选项（安全但平庸）
3. 平衡选项（有得有失）
4. 隐藏选项（需要特定条件才显示）
5. 道德极端选项（需要特定道德值）
```

### 3.4 设计后果

**每个选项必须包含：**

```
1. 即时后果：属性变化
2. 延迟后果：设置flag
3. 人物关系：影响关键人物好感
4. 结果描述：文本描述
```

## 四、选项设计规则

### 4.1 选项显示条件

```typescript
// 某些选项需要特定条件才显示
showConditions: {
  // 属性门槛
  attributes: { 武略: { min: 60 } },
  
  // 道德门槛
  hidden: { 道德值: { max: 30 } },  // 只有道德值低才能选
  
  // 野心门槛
  hidden: { 野心值: { min: 70 } },  // 只有野心值高才能选
  
  // 标记门槛
  flags: { has: ["前置事件flag"] }
}
```

### 4.2 选项文本设计

**选项文本要简洁明了，体现代价。**

```
❌ 错误示例：
选项A：帮助百姓（太模糊）
选项B：不帮助百姓（太简单）

✅ 正确示例：
选项A：开仓放粮赈灾（财帛-20，民望+25，可能被弹劾）
选项B：按章办事，上报朝廷（安全，但民怨沸腾）
选项C：虚报灾情，中饱私囊（财帛+30，道德值-20，risky）
```

### 4.3 选项代价机制

**每个选项必须有至少一个负面效果。**

```
类型1：资源代价
- 财帛-20
- 体质-10

类型2：态度代价
- 圣眷-15
- 清议-20

类型3：风险代价
- 设置flag:可能被弹劾
- 设置flag:可能被牵连

类型4：道德代价
- 道德值-15
- 良心谴责
```

## 五、属性变化逻辑

### 5.1 行为类型 → 属性变化映射

| 行为类型 | 个人能力变化 | 五方态度变化 | 隐藏属性变化 |
|---------|-------------|-------------|-------------|
| **清廉正直** | 文韬+ 理政+ | 圣眷+ 清议+ 士绅+ 民望+ | 道德值+ 欲望值- |
| **贪污受贿** | 财帛+ | 清议- 民望- 圣眷-(风险) | 道德值- 欲望值+ |
| **结党营私** | 理政+ | 中官+ 清议- | 道德值- 野心值+ |
| **直言进谏** | 文韬+ | 圣眷± 清议+ | 道德值+ |
| **军事行动** | 武略+ 体质- | 圣眷+ 民望± | 野心值+ |
| **体恤百姓** | 财帛- | 民望+ 清议+ | 道德值+ |

### 5.2 属性变化幅度标准

| 变化幅度 | 数值范围 | 适用场景 |
|---------|---------|---------|
| **微小** | ±1~5 | 日常小事、轻微影响 |
| **一般** | ±6~15 | 常规事件、正常选择 |
| **重大** | ±16~30 | 关键事件、重要选择 |
| **极端** | ±31~50 | 历史节点、生死抉择 |

### 5.3 属性变化示例

```typescript
// 示例1：清廉正直的选择
effects: {
  attributes: { 文韬: 5, 理政: 5 },
  gameState: { 圣眷: 10, 清议: 15, 民望: 10 },
  hidden: { 道德值: 10, 欲望值: -5 }
}

// 示例2：贪污受贿的选择
effects: {
  attributes: { 财帛: 30 },
  gameState: { 清议: -20, 民望: -15 },
  hidden: { 道德值: -20, 欲望值: 10 },
  flags: { add: ["贪污记录"] }
}

// 示例3：军事行动的选择
effects: {
  attributes: { 武略: 10, 体质: -15 },
  gameState: { 圣眷: 15, 民望: 10 },
  hidden: { 野心值: 10 }
}
```

## 六、蝴蝶效应机制

### 6.1 标记系统

**通过flag标记实现蝴蝶效应。**

```typescript
// 当前选择设置flag
effects: {
  flags: { add: ["结交阉党"] }
}

// 未来事件检查flag
conditions: {
  flags: { has: ["结交阉党"] }
}

// 未来事件移除flag
effects: {
  flags: { remove: ["结交阉党"] }
}
```

### 6.2 连锁事件设计

```
事件A → 设置flag_A → 影响事件B的触发
事件B → 设置flag_B → 影响事件C的选项
事件C → 设置flag_C → 影响事件D的后果

示例：
1628年：结交阉党 → flag:阉党关系
1629年：诛杀魏忠贤 → 检查flag:阉党关系 → 可能被牵连
1630年：温体仁清算 → 检查flag:阉党关系 → 可能被清洗
```

### 6.3 历史改变机制

```typescript
// 历史改变状态
historicalChanges: {
  yuanChonghuan: 'dead' | 'exiled' | 'alive',
  liZicheng: 'dead' | 'pacified' | 'emperor' | 'alive',
  chongzhen: 'dead' | 'nanjing' | 'alive',
  songjin: 'lost' | 'won' | 'avoided',
  currentTimeline: 'original' | 'liaodong' | 'bandit' | 'nanjing' | 'reform' | 'usurper'
}

// 检查历史改变状态
conditions: {
  historicalFlags: {
    yuanChonghuan: 'alive'
  }
}
```

## 八、常见错误与避免

### 8.1 属性变化过于简单

```
❌ 错误：
effects: { attributes: { 财帛: 10 } }

✅ 正确：
effects: {
  attributes: { 财帛: 10, 文韬: -5 },
  gameState: { 清议: -10 },
  hidden: { 道德值: -5 }
}
```

### 8.2 选项没有代价

```
❌ 错误：
选项A：清廉正直 → 所有属性+10

✅ 正确：
选项A：清廉正直 → 清议+15，财帛-20，可能被孤立
```

### 8.3 忽略五方态度联动

```
❌ 错误：
提升清议时，中官没有变化

✅ 正确：
提升清议+15时，中官-5（清流与宦官冲突）
```

### 8.4 蝴蝶效应缺失

```
❌ 错误：
选择后没有任何标记，不影响未来事件

✅ 正确：
effects: {
  flags: { add: ["结交阉党"] }
}
// 未来事件检查此flag
```

### 8.5 历史改变过于容易

```
❌ 错误：
改变袁崇焕命运：只需要选择"营救"

✅ 正确：
改变袁崇焕命运：
1. 需要：官职给事中/御史
2. 需要：文韬>70，理政>50
3. 需要：圣眷>40
4. 需要：在特定时间窗口（1630年1-3月）
5. 成功率：30-50%
```
