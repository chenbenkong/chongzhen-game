## 三、属性对事件选项的影响

### 3.1 属性门槛机制

**基本原则**：
- 高属性解锁特殊选项（优势选项）
- 低属性限制某些选项（劣势限制）
- 属性值影响成功率

#### 示例1：军事行动事件

```typescript
{
  id: "military_001",
  title: "流寇入境",
  description: "高迎祥、李自成部进入辖区，你该如何应对？",
  choices: [
    {
      id: 'c1',
      text: '主动出击，剿灭流寇',
      // 高武略选项：成功率更高
      showConditions: {
        attributes: { 武略: { min: 70 } }
      },
      effects: {
        attributes: { 武略: 10, 体质: -10 },
        gameState: { 圣眷: 15, 民望: 10 },
        // 成功率计算：武略/100 * 80%
        successRate: '武略 * 0.8'
      }
    },
    {
      id: 'c2',
      text: '固守城池，等待援军',
      // 中等武略选项：安全但平庸
      showConditions: {
        attributes: { 武略: { min: 40 } }
      },
      effects: {
        attributes: { 体质: -5 },
        gameState: { 民望: -10 }
      }
    },
    {
      id: 'c3',
      text: '坚壁清野，消耗敌人',
      // 低武略选项：消耗大但有效
      effects: {
        attributes: { 财帛: -25, 体质: -5 },
        gameState: { 民望: 10 }
      }
    },
    {
      id: 'c4',
      text: '开城投降',
      // 极低武略选项：保命但遗臭万年
      showConditions: {
        attributes: { 武略: { max: 20 } }
      },
      effects: {
        gameState: { 圣眷: -50, 清议: -50 },
        hidden: { 道德值: -30 },
        special: { type: 'ending', value: '降贼结局' }
      }
    }
  ]
}
```

### 3.2 成功率计算机制

**基本公式**：
```
成功率 = 基础成功率 × (属性值/100) × 随机因子

示例：
- 基础成功率：60%
- 属性值：文韬=80
- 随机因子：0.8-1.2
- 最终成功率 = 60% × 0.8 × 1.0 = 48%
```

**属性影响表**：

| 属性值 | 成功率修正 | 说明 |
|--------|-----------|------|
| 90-100 | +30% | 极高属性，大幅提升成功率 |
| 70-89 | +20% | 高属性，提升成功率 |
| 50-69 | +10% | 中等属性，略微提升 |
| 30-49 | 0% | 正常成功率 |
| 10-29 | -20% | 低属性，降低成功率 |
| 0-9 | -50% | 极低属性，大幅降低成功率 |
| <0 | 无法选择 | 属性<0，游戏结束 |

### 3.3 属性联动机制

**五方态度联动**：

```typescript
// 当清议变化时，自动触发中官反向变化
function updateAttitude(gameState, attitude, change) {
  gameState[attitude] += change
  
  // 联动规则
  if (attitude === '清议') {
    // 清议↑ → 中官↓（清流与宦官冲突）
    gameState.中官 -= Math.floor(change * 0.3)
  }
  
  if (attitude === '中官') {
    // 中官↑ → 清议↓（宦官与清流冲突）
    gameState.清议 -= Math.floor(change * 0.3)
  }
  
  if (attitude === '民望') {
    // 民望↑ → 士绅↓（可能冲突）
    if (change > 0) {
      gameState.士绅 -= Math.floor(change * 0.2)
    }
  }
  
  // 确保属性在合理范围内
  // 注意：属性可以为0，但不能<0（否则游戏结束）
  for (let key in gameState) {
    if (gameState[key] < -10) gameState[key] = -10  // 允许短暂<0，但立即结束游戏
    if (gameState[key] > 100) gameState[key] = 100
  }
}
```

## 四、属性恢复与补救机制

### 4.1 自然恢复机制

| 属性 | 自然恢复 | 触发条件 |
|------|----------|----------|
| **体质** | +5/月 | 休息、养病 |
| **圣眷** | -1/月 | 崇祯多疑，自然下降 |
| **国势** | -0.5/月 | 大势所趋，自然下降 |

### 4.2 主动恢复机制

#### 【财帛恢复】
- **俸禄**：按品级固定收入（+5/月）
- **赏赐**：皇帝/上级赏赐（不定）
- **田产**：购置田产收租（+1/10亩/月）
- **借贷**：向亲友/钱庄借贷（有利息）
- **变卖**：变卖资产（一次性收入）

#### 【体质恢复】
- **休息**：每回合+5（消耗行动点）
- **养病**：每回合+10（消耗财帛-10）
- **药物**：每回合+15（消耗财帛-30）
- **补品**：每回合+20（消耗财帛-50）

#### 【圣眷恢复】
- **办事得力**：完成任务（+5~15）
- **直言进谏**：短期-，长期+（风险高）
- **贿赂宦官**：通过中官影响（需要中官>50）
- **立功**：军事/政治大功（+20~30）

#### 【清议恢复】
- **撰写文章**：发表文章（+5~10）
- **参与文会**：参加文会（+5）
- **清廉正直**：拒绝贿赂（+10~20）
- **弹劾奸佞**：弹劾贪官（+15~25）

#### 【其他属性恢复】
- **文韬**：读书、游学、撰写文章
- **理政**：处理政务、完成任务
- **武略**：边疆任职、军事行动
- **中官**：贿赂宦官、结交宦官
- **士绅**：体恤士绅、减轻赋税
- **民望**：赈灾、减税、清廉

### 4.3 应急补救机制

#### 【濒临破产（财帛=0）】
触发"濒临破产"事件：
- **向亲友借贷**：无利息，但可能被拒绝
- **向钱庄借贷**：利息10%/月，到期不还后果严重
- **向高利贷借贷**：利息30%/月，极度危险
- **变卖资产**：一次性获得财帛，但失去资产
- **如果无法补救**：下回合财帛<0 → 破产结局

#### 【濒临死亡（体质=0）】
触发"濒临死亡"事件：
