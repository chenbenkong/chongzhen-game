## 一、核心概念

### 1.1 游戏定位
- **类型**：明末历史向文字策略模拟游戏
- **风格**：黑色幽默标题 + 严肃残酷内核
- **核心体验**：人在时代洪流中的挣扎与选择
- **设计哲学**：没有正确答案，只有局势判断

### 1.2 核心主题

**三大核心张力**：
1. **个人命运 vs 时代洪流**：个人选择在大势面前的无力感
2. **理想 vs 现实**：忠、能、狠、忍、名、命之间的取舍
3. **短期生存 vs 长远影响**：每一步都在钢丝上行走

### 1.3 时间线设定
- **起始年份**：崇祯元年（1628年）
- **玩家年龄**：16岁（准备科考）
- **游戏跨度**：16年（1628-1644，至明朝灭亡）
- **关键历史节点**：20个

### 1.4 历史改变系统核心理念

**三大原则**：
1. **蝴蝶效应**：小改变可能引发大波澜，但方向不可控
2. **代价机制**：改变历史需要牺牲，没有免费的午餐
3. **多重可能性**：同一历史节点有多种改变路径

**历史改变难度分级**：

| 难度等级 | 改变对象 | 所需条件 | 成功率 |
|---------|---------|---------|--------|
| ★ | 个人命运 | 高属性/正确选择 | 60-80% |
| ★★ | 地方事件 | 特定官职/时机 | 40-60% |
| ★★★ | 朝中决策 | 高位+圣眷+时机 | 30-50% |
| ★★★★ | 历史人物命运 | 极端条件+连锁反应 | 15-30% |
| ★★★★★ | 历史大势 | 多节点成功+运气 | 5-15% |

## 四、游戏阶段与历史节点（1628-1644）

### 4.1 第一阶段：寒窗苦读（16-20岁）1628-1630

#### 节点1：崇祯登基（1628年正月）
**历史背景**：天启帝驾崩，信王朱由检即位

**玩家处境**：
- 寒门/诗文：正在准备科举，听闻新帝登基，朝局将变
- 缙绅/世家：家族开始布局，寻找新朝靠山

**关键选择**：
- 是否参与"哭庙"（悼念天启）
- 是否上书"新政建议"
- 是否结交阉党余孽（risky but potentially rewarding）

**蝴蝶效应**：
- 结交阉党 → 后被牵连（温体仁清算）
- 上书新政 → 被崇祯注意（圣眷+5）
- 保持沉默 → 安全但无亮点

---

#### 节点2：诛杀魏忠贤（1628年秋）
**历史背景**：崇祯清算阉党，魏忠贤自缢

**玩家处境**：
- 如果之前结交阉党：必须切割关系
- 如果之前弹劾阉党：获得政治资本

**关键选择**：
- **A. 积极参与清算**：清议+15，但树敌（阉党余孽仇恨+50）
- **B. 保持中立**：安全，但无收益
- **C. 暗中保护某些阉党**：获得王德化等太监好感（后续宫中有人），但 risky

**连锁反应**：
- 积极参与 → 被东林党接纳（清议+20）
- 保护阉党 → 中官关系网建立

---

#### 节点3：己巳之变（1629年冬）
**历史背景**：皇太极绕道蒙古，突破长城，围困北京

**详细事件链**：

**事件3.1：边警初现（1629年八月）**
- 蒙古部落异动，有商人报告后金可能在集结
- 玩家如果官职在兵部/言官，可以上疏预警
- 需要：武略>60，理政>40
- 成功：获得"先见之明"声望，圣眷+10

**事件3.2：蓟镇失守（1629年十月）**
- 皇太极突破喜峰口
- 玩家如果在地方，可以选择是否勤王
- 提前勤王：不等诏令，主动率军入卫
  - 风险：可能被疑为"擅离职守"
  - 需要：理政>50（朝中有人为你说话）

**事件3.3：北京保卫战（1629年十一月）**
- 各路勤王军到达
- 袁崇焕被猜忌
- 玩家可以选择：
  - **支持袁崇焕**：risky，但可能救他（开启救袁线）
  - **保持沉默**：安全
  - **弹劾袁崇焕**：获得崇祯好感（圣眷+15），但清议唾弃（清议-20）

**事件3.4：袁崇焕下狱（1629年十二月）**
- 袁崇焕被捕
- 玩家可以选择是否营救
- 营救需要：言官身份 + 文韬>70 + 理政>50 + 圣眷>40

### 4.2 第二阶段：初入仕途（21-28岁）1630-1637

#### 节点4：袁崇焕之死（1630年八月）
**历史背景**：袁崇焕被凌迟处死

**改变路径A：力保袁崇焕**
- **触发条件**：官职给事中/御史，时间1630年1-3月，文韬>70，理政>50，圣眷>40
- **行动链**：
  1. 上疏辩护：指出反间计，请求慎重
     - 成功：袁崇焕改判流放海南
     - 失败：被牵连，圣眷-20，廷杖风险
  2. 联络东林：发动清议施压（需要清议>60）
     - 效果：皇帝压力+，但猜忌++
  3. 私下进言：利用召对机会面陈（需要圣眷>60，理政>70）

### 5.1 事件分类

| 类别 | 数量 | 内容 |
|------|------|------|
| 科举阶段 | 20个 | 县试到殿试全流程 + 座师选择 + 同年关系 |
| 地方任职 | 30个 | 刑名、钱粮、教化、军事、突发、交际 |

| 情感事件 | 20个 | 正妻、妾室、红颜知己相关 |
| 灰色选择 | 20个 | 荒淫、贪污、叛乱、暗杀等 |

**基础事件库：100个（来自原始内容库）**
**扩展事件库：40个（情感20 + 灰色20）**
**总计：140+个详细事件**

### 5.2 事件设计原则

1. **没有正确答案**：每个选择都有代价
2. **连锁反应**：当前选择影响未来事件
3. **蝴蝶效应**：小改变引发大波澜
4. **历史惯性**：改变历史很难，大势难违
5. **多重后果**：一个选择影响2-3个属性

### 13.3 游戏循环

```javascript
// 每回合流程
gameLoop: {
  // 1. 时间推进
  turn++,
  month++,
  if (month > 12) { year++; month = 1; },
  
  // 2. 自然属性变化
  character.age += 1/12,  // 年龄增长
  gameState.国势 -= 0.5,  // 国势自然下降
  gameState.圣眷 -= 1,   // 圣眷自然下降（崇祯多疑）
  
  // 3. 检查触发事件
  availableEvents = checkEventConditions(allEvents, gameState),
  
  // 4. 显示事件
  if (availableEvents.length > 0) {
    currentEvent = selectEvent(availableEvents),
    showEvent(currentEvent)
  } else {
    // 5. 日常决策
    showDailyChoices()
  },
  
  // 6. 检查结局条件
  if (checkEndingConditions()) {
    showEnding()
  }
}
```
